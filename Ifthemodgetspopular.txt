If you ever wanted to enable strict security to verify they bought the game, we would do it using **Mojang's Official Session Servers**. Here is how the mod would do it behind the scenes:

1. **The Handshake:** When the mod connects to your Hugging Face backend, your backend would generate a random "Server ID" (like `abc123xyz`) and send it to the player.
2. **Client Approval:** The Java mod would take that Server ID and silently send a request to `sessionserver.mojang.com`, saying: *"Hi Mojang, I am GTSIM and I am joining server abc123xyz."*
3. **Backend Verification:** Your Hugging Face server would then ask Mojang: *"Did the real GTSIM just join abc123xyz?"*
4. **The Verdict:** Mojang checks their database. If it's a cracked account, Mojang says *"No"*, and your backend immediately kicks them. If it's a real premium account, Mojang says *"Yes, here is their official UUID"*, and your backend lets them in.

This is the exact same method that real Minecraft servers (like Hypixel) use to block cracked players. So yes, it is 100% possible to add!

**Global Like System with Firebase**

1. **Global Server Database:** We update the Python backend to keep a permanent database of songs and their "Like" counts using a free **Firebase Realtime Database**. Every time a song gets liked, its score goes up globally for everyone using the mod.
2. **Secure Connection via Secrets:** The Hugging Face Space uses a hidden **"Secret"** to store the database password. This guarantees the database is entirely private and hackers cannot edit it, even if the Space's code is public.
3. **Liking a Song In-Game:** We add a keybind (`L`). When pointing at a player who is listening to a song, pressing the keybind tells the server: *"Add 1 like to [Song Name]"*.
4. **Anti-Cheat:** The Python backend tracks Minecraft Usernames and ensures a player can only like a specific song once, preventing spam.
5. **Global Leaderboard:** We add an in-game command (`/spotify top`) or a GUI button that fetches the top 10 most-liked songs across the entire mod's history.
